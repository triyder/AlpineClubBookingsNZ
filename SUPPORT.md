# Support

## Using This Repository

Use GitHub issues for reproducible bugs, documentation gaps, and feature ideas
related to the open-source reference code.

Before opening an issue, check:

- `README.md` for setup and common commands
- `docs/ARCHITECTURE.md` for system design
- `docs/STAGING_ACCESSIBILITY.md` for staging verification
- existing open and closed issues

## Production Support

This public repository is not the support channel for any live Tokoroa Alpine
Club service. Production users should contact the club through its normal
support channels.

Do not post personal information, payment details, accounting data, logs with
secrets, or live customer records in public issues.
